## 🌟 ¿Qué es Potatously Optimized?

**Potatously Optimized** es un modpack de Minecraft Fabric diseñado para jugadores hispanohablantes que buscan:

- ⚡ **Máximo rendimiento** - Juega con FPS altos incluso en computadoras modestas
- 🎨 **Gráficos mejorados** - Shaders, iluminación dinámica y efectos visuales opcionales
- 🔧 **Optimización inteligente** - Configuraciones pre-ajustadas para el mejor equilibrio
- 🌍 **Soporte en español** - Documentación, comunidad y soporte completamente en español
- 🎮 **Compatible con servidores** - Funciona tanto en singleplayer como multiplayer

## 📋 Requisitos

- **Minecraft**: Java Edition 1.21 o superior
- **Loader**: Fabric Loader (se instala automáticamente)
- **RAM**: Mínimo 4GB asignados (recomendado 6GB)
- **Java**: Java 21 o superior

---

## 🚀 Instalación

### Método 1: Modrinth App (Recomendado)

1. Descarga [Modrinth App](https://modrinth.com/app)
2. Busca "Potatously Optimized" en la app
3. Haz clic en **Instalar**
4. ¡Listo para jugar!

### Método 2: Manual

1. Instala [Fabric Loader](https://fabricmc.net/use/)
2. Descarga el modpack desde [Modrinth](https://modrinth.com/modpack/potatously-optimized)
3. Extrae los mods en tu carpeta `.minecraft/mods`
4. Inicia Minecraft con el perfil de Fabric

### Método 3: Otros Launchers

- **Prism Launcher**: Añadir desde Modrinth
- **ATLauncher**: Importar modpack
- **MultiMC**: Importar modpack

## ⚙️ Configuración Recomendada

### Para PCs Modestas (Papas 🥔)
```
RAM asignada: 4GB
Distancia de renderizado: 8-12 chunks
VSync: Activado
FPS límite: 60
Shaders: Desactivados
```

### Para PCs Potentes
```
RAM asignada: 6-8GB
Distancia de renderizado: 16-24 chunks
VSync: Desactivado
FPS límite: Sin límite
Shaders: Activados (recomendamos Complementary)
```
